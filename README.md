# 🧮 Calculadora de IMC

Aplicação web para calcular o **Índice de Massa Corporal (IMC)** de forma simples e rápida, com classificação do resultado de acordo com as faixas estabelecidas pela OMS.

---

## 📋 Sobre o Projeto

O IMC é uma medida internacional usada para avaliar se uma pessoa está no peso ideal em relação à sua altura. Esta calculadora permite que o usuário insira seu **peso (kg)** e **altura (m)** e receba instantaneamente o valor do IMC junto com uma descrição da classificação.

---

## ✨ Funcionalidades

- Inserção de peso em kg e altura em metros
- Cálculo automático do IMC ao clicar em "Calcular"
- Exibição do valor do IMC com duas casas decimais
- Classificação do resultado nas seguintes faixas:

| IMC | Classificação |
|-----|---------------|
| Abaixo de 18,5 | Abaixo do peso |
| 18,5 – 25,0 | Peso ideal |
| 25,0 – 30,0 | Sobrepeso |
| 30,0 – 35,0 | Obesidade moderada |
| 35,0 – 40,0 | Obesidade severa |
| Acima de 40,0 | Obesidade mórbida |

- Link para mais informações sobre o IMC

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Marc0sbernardo/Calculadora-IMC.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Calculadora-IMC
   ```

3. Abra o arquivo `index.html` no seu navegador.

> Não é necessário instalar dependências ou configurar servidor — basta abrir o arquivo diretamente!
